ACTIVIDADES EXTRA DE LA CLASE 02

ACTIVIDAD 01

CONSIGNA:

Solicitar al usuario un nombre. Si el mismo es igual a otro nombre almacenado en una variable, realizar una salida por alerta con el mensaje "Fui yo". Caso contrario, la salida será "Yo no fui".




ACTIVIDAD 02

CONSIGNA:

Solicitar al usuario una tecla. Si se presiona "y" (minúscula), o "Y" (mayúscula), realizar una salida por alerta con el mensaje "Correcto". Caso contrario, la salida será "Error".




ACTIVIDAD 03

CONSIGNA:

Solicitar al usuario un número. Si el valor está entre 1 y 4, efectuar una de las siguientes salidas según corresponda: "Elegiste a Homero" si es 1. "Elegiste a Marge" si es 2. "Elegiste a Bart" si es 3. "Elegiste a Lisa si es 4.




ACTIVIDAD 04

CONSIGNA:

Solicitar al usuario un número. Si el valor está entre dos números, efectuar una de las siguientes salidas, según corresponda: "Presupuesto bajo" si está entre 0 y 1000. "Presupuesto medio" si está entre 1001 y 3000. "Presupuesto alto" si es mayor a 3000.




ACTIVIDAD 05

CONSIGNA:

Solicitar al usuario cuatro productos de almacén. Si todos los elementos fueron cargados, realizar una única salida compuesta por el listado de productos. Caso contrario, la salida será "Error: Es necesario cargar todos los productos".
